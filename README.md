###Noise Generator

**Requirements:** GSL library (included) 
**Limitations:** none  

![Noise Generator GUI](noise-generator.png)  

<!--start-->
This module continuously generates Gaussian white noise computed using the Box-Muller method. 
<!--end-->

####Output Channels
1. output(0) – “Noise Waveform”  

####Parameters
1. Mean  
2. Variance
